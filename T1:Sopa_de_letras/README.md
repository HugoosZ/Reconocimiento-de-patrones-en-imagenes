## Requisitos

El programa está desarrollado en **Python 3** y utiliza las siguientes librerías:

- `numpy`
- `pandas`
- `matplotlib`
- `opencv-python`
- `scikit-image`

Puedes instalarlas con:

```bash
pip install numpy pandas matplotlib opencv-python scikit-image
```
